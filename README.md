## Installation

1. Clone this repository:

```
git clone https://github.com/yourusername/spaceblaster.git
```
2. Navigate to the project directory:
```
cd spaceblaster
```
3. Compile the project:
```
make
```
4. Run the game
```
./spaceblaster
```
